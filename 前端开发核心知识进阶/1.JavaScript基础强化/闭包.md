## 闭包

```javascript
// 暂时性死区会在块级作用域中发生,在函数参数的默认值处也会发生
function foo(arg1) {
    let arg1;
}

// 等效于
function foo(arg1) {
    var arg1
    let arg1;
}

foo('args');
```

null 和 undefined 的区别

undefined 作为参数传给函数时,如果函数有对应的默认值,则会取默认值
null 作为参数传给函数时,则会替代默认值

JavaScript 执行: 1.代码预编译(变量声明,声明提升,非函数表达式的函数声明提升) 2.代码编译

```javascript
// 这个非函数表达式(会发生函数声明提升)
function test() {
  console.log('test1')
}

// 这个是函数表达式
var test = function() {
  console.log('test2')
}
```

和

```javascript
var test = function() {
  console.log('test2')
}

// 这个非函数表达式(会发生函数声明提升)
function test() {
  console.log('test1')
}
```

等效于

```javascript
var test
test = function() { 
  console.log('test1')
}

test = function() {
  console.log('test2')
}
```
